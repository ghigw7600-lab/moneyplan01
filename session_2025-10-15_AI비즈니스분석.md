## 💬 대화 주요 내용

### Q1: 웹 Claude vs VSCode Claude 차이?
**A**:
- 웹 Claude = 대화만 (코드 실행 불가)
- VSCode Claude = 실제 파일 제어 + 명령어 실행
- VSCode가 "실제 작업하는 동료 개발자"

### Q2: 대화 기억은?
**A**:
- 대화 기억 = 휘발성 (새 세션 시 사라짐)
- 파일 저장 = 영구적 (CLAUDE.md 등)
- 중요한 건 파일로 저장 필요

### Q3: 동시 진행 전략?
**A**:
- 웹사이트 (빠른 수익) + 영상 (추가 수익) + 챗봇 (고수익)
- 리스크 분산 + 시너지 효과
- 3가지 결합 → 토탈 솔루션 (고단가)
