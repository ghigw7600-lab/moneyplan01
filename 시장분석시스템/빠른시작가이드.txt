═══════════════════════════════════════════════════════════

실행 방법:

추천 기준: (완화됨!)
  - 신뢰도 60% 이상 (기존 70% → 60%)
  - RSI 55 이하 (기존 35 → 55)
  - 매수/강력매수 신호

결과 예시:

  [1] 현대차 (005380.KS)
     💰 현재가: 285,000원
     🎯 신뢰도: 65%
     📊 RSI: 53.8
     📈 신호: buy

가상화폐 API 제한:
  - CoinGecko 무료 API는 요청 제한 있음
  - 자동으로 3초 딜레이 추가됨
  - 주식만 스캔하려면 "1번" 선택

📊 추천 기준 변경 사항
═══════════════════════════════════════════════════════════

기존 (너무 엄격):
  ❌ 신뢰도 70% 이상
  ❌ RSI 35 이하
  → 추천 종목 거의 없음

변경 (현실적):
  ✅ 신뢰도 60% 이상
  ✅ RSI 55 이하
  → 더 많은 종목 추천!

왜 변경했나요?
  - 현재 시장이 강세장이라 RSI가 전반적으로 높음
  - 과매도(RSI 30 이하)만 찾으면 종목이 거의 없음
  - RSI 55 정도면 중립~약간 과매수 구간으로 적절함

🐛 문제 해결
═══════════════════════════════════════════════════════════

Q1. 웹 대시보드가 안 열려요
A1. http://localhost:5001 접속 (5000이 아닌 5001!)

Q2. 추천 종목이 0개예요
A2. 정상! 시장 상황에 따라 매수 신호가 없을 수 있음
    - 기준 완화했으니 이제 더 많이 추천됨
    - 가상화폐는 API 제한으로 건너뛰고 주식만 스캔 권장

Q3. CoinGecko API 오류 (429 Too Many Requests)
A3. 무료 API 요청 제한 초과
    - "1. 한국 주식 스캔"만 선택하세요
    - 또는 30분 후 다시 시도

Q4. Flask 설치 오류
A4. 이미 설치 완료! 웹대시보드시작.bat 다시 실행하세요
